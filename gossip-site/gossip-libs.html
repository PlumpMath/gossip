The GOSSIP Simulation Environment

gossip-lib-*

The following libraries are available directly from the Gossip project.
  • gossip-lib-skel, a `skeleton' library that shows how the infrastructure of a real library might look like.
  • gossip-lib-misc, a library of miscellaneous goodies like type converters, arithmetic blocks, bit shifters, statistics blocks, sources, destinations etc.
  • gossip-lib-radio, with mixers, oscillators, modulators, etc.
  • gossip-lib-fec, forward error correction blocks for codes like Hamming, BCH, Golay, Reed-Solomon, Viterby, etc.
